ARP: Address Resolution Protocol:


ARP, or Address Resolution Protocol, is a network protocol used to map Internet Protocol (IP) addresses to a physical, MAC, address that is recognised

in the network. A table is used to keep track of which machine addresses have which IP address.

When a packet arrives at a router on its way to host A, the router asks the ARP program to figure out where the packet is going. The ARP program checks its ARP table to find the MAC address that

corresponds to the destination IP address found in the packet. If it finds a MAC address, it tells the router which sends the packet on its way.

If no such MAC address is found, the ARP program broadcasts all the members of the network to ask if anyone knows who the receiver of the packet is. If a machine recognises that IP address as its own,

then it sends a message to the router to inform it so. The ARP table is then updated accordingly.
